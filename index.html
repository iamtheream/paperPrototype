PAPERPROTO

Paper prototyping is a efficient way to quickly express your vision in a way that is in perspective to real use while promoting an agile workflow. Inspired by the Internet famous IronPhone we thought there was surely a 3D printable model to use for paper prototyping, but there was none. So, we made it. The .ZIP file contains the .STL and .PDF so you can get started quickly mocking up awesome mobile experiences.

PERSPECTIVE

Putting perspective and actually in mobile size constraints is really important. It’s the first step in crafting a proper digital experience.

INTERACTION DESIGN

The ability to quickly study navigation and animations by having slots in the top and bottom. The templates allows for quick content management and design.

AGILE WORKFLOW

Drawing has a different psychology than jumping right into digital work. It's easily disposable. Showing teammates paper prototypes allows for easy changes and input.

PRINT

Once you have downloaded the
.ZIP file. Extract the folder and load the .STL file onto your 3D printer, or purchase it from Shapeways.com. Once the print has started, send the .PDF file to your paper printer and make sure it’s set for tabloid.

CUT

While the phone is 3D printing, you can start cutting out the templates. Be sure to cut along the dotted lines and around the top tab. This is what will be used for gripping the paper to simulate animation during prototyping.

SKETCH

The phone should almost be done printing and you can start sketching. Be sure to use the blue tap state size guide and lines to indicate native iPhone UI elements screen sizes.

PROTOTYPE

Remove the phone from the printer. The side slots for horizontal scrolling sites or hidden NAV content and the top and bottom slots for vertical scrolling pages with lots of content.
